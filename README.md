# 💰 PaiseKamao.in – Website

## अपनी Website कैसे चलाएं?

### Local Computer पर:
```bash
pip install flask
python app.py
# Browser में खोलो: http://localhost:5000
```

### Free Hosting (Render.com):
1. GitHub पर account बनाओ
2. यह files upload करो
3. render.com पर जाओ → New Web Service → GitHub connect करो
4. Start Command: `gunicorn app:app`
5. 5 मिनट में live हो जाएगी!

### Domain (paisekamao.in):
- GoDaddy.com या Namecheap.com पर जाओ
- "paisekamao.in" search करो
- .in domain ₹500-₹800/साल में मिलता है
- Render.com के साथ custom domain connect करो

### Traffic कैसे लाएं?
1. Google पर submit करो: search.google.com/search-console
2. हर page पर SEO keywords डालो
3. YouTube पर video बनाओ और website link दो
4. Quora, Reddit, Facebook Groups में share करो
5. WhatsApp groups में share करो

## Files:
- `app.py` → Main Python Flask app
- `requirements.txt` → Dependencies
